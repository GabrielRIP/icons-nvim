## Uso

para [packer](https://github.com/wbthomason/packer.nvim):

```lua
  use {'grChad/icons-nvim'}

  -- Para usarlo:
  local icons = require('i-nvim')
```

Para [lazy](https://github.com/folke/lazy.nvim)

```lua
   {
      'grChad/icons-nvim',
      lazy = true,
   },

  -- Para usarlo:
  local icons = require('i-nvim')
```
